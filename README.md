🏠 ROOMIT Frontend
룸메이트 매칭과 주거 공간 탐색을 결합한 웹 서비스 ROOMIT의 프론트엔드 프로젝트입니다.  
사용자는 자신의 생활 패턴에 맞는 룸메이트를 찾고, 다양한 주거 공간 정보를 탐색할 수 있습니다.
---
📌 프로젝트 소개
ROOMIT은 1인 가구 증가로 인해 발생하는
높은 주거 비용
생활 패턴 불일치
사회적 고립 문제
를 해결하기 위해 기획된 서비스입니다.
단순한 방 검색이 아닌,  
👉 "나와 맞는 사람 + 나에게 맞는 공간"을 함께 찾는 경험을 제공합니다.
---
🚀 주요 기능
1. 🏘 주거 공간 탐색
다양한 주거 공간 리스트 조회
카드 UI 기반 정보 제공
필터를 통한 조건별 검색
지역
가격
면적 등
---
2. 👥 룸메이트 매칭
사용자 프로필 기반 룸메이트 탐색
카드 형태로 직관적인 정보 확인
성향 기반 매칭 UI 제공
---
3. 💬 채팅 기능
사용자 간 커뮤니케이션 지원
룸메이트 매칭 이후 자연스러운 대화 연결
---
4. 🔐 인증 및 접근 제어
로그인 사용자 / 비로그인 사용자 분리
AuthGuard, GuestGuard를 통한 라우팅 제어
---
5. 🎯 UI/UX 중심 설계
컴포넌트 단위 구조 설계
재사용 가능한 UI 구성
사용자 흐름 중심 화면 구성
---

⚙️ 기술 스택
Frontend: React
Build Tool: Vite
Styling: CSS
상태 관리: React Hooks
라우팅 보호: Custom Guard
---
💡 설계 특징
1. 컴포넌트 기반 구조
UI를 기능 단위로 분리하여 유지보수성 향상
재사용 가능한 카드 컴포넌트 활용
2. 역할 분리
인증 로직과 UI 분리
데이터 표시와 필터링 UI 분리
3. 확장 고려 구조
채팅, 매칭, 공간 탐색 기능을 독립적으로 확장 가능
향후 API 연동 및 상태 관리 라이브러리 도입 용이
